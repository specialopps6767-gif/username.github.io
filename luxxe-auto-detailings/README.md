## 🚀 How to View

### Option 1: Open Directly
1. Double-click `index.html` in the folder
2. Your default browser will open the website

### Option 2: Local Server (Recommended)
```bash
# Navigate to folder
cd luxxe-auto-detailings

# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js
npx http-server
```
Then visit: `http://localhost:8000`

### Option 3: VS Code Live Server
1. Install "Live Server" extension
2. Right-click `index.html`
3. Select "Open with Live Server"

### Change Pricing
Update the pricing section in `index.html` with new prices

### Update Gallery Images
Replace image URLs in the gallery section:
```html
<img src="YOUR_IMAGE_URL" alt="Description">
```

### Customize Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-gold: #d4af37;
    --dark-bg: #0a0a0a;
    /* ... other variables ... */
}
```
